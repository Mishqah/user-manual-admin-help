Create Social media link

Once you have selected the settings button the company information screen will be displayed where the social links will be displayed in a list format. To create social media links, you will select the add button underneath the social media heading. Once selected an input field will display where you will then paste or enter the social media link. If there is more than one social media link you will select the add button next to the social media link and once another input field has displayed, you will enter or paste the second social media link. To add more you can simply select the add button explained above. Update Social media link

Once you have selected the settings button the company information screen will be displayed where the social links will be displayed in a list format. To update social media links, you will select the edit button next to the social media link you wish to update. Once selected the input field with the social media links will display where you will then paste or enter the updated social media links and select the save button. A pop-up message will appear stating that it was successfully saved, and you will notice that the updated social media link will be displayed amongst the list.
