View Sale

When accessing the Sales part of the system the admin will need to be logged in and navigate to the admin home screen. In the admin home screen, there will be numerous tabs containing links to the various subsystems. The admin will have to click on the sales button to gain access to the sales management screen. On the Sales Management Screen the Admin will click on the Input Field to search for the desired client’s details using the entered criteria.

Cancel Sale

When accessing the Sales part of the system the admin will need to be logged in and navigate to the admin home screen. In the admin home screen, there will be numerous tabs containing links to the various subsystems. The admin will have to click on the sales button to gain access to the sales management screen. On the Sales Management Screen the Admin will click on the Cancel Sale button to delete the selected sale of the client.

 

Regenerate Sale Invoice

When accessing the Sales part of the system the admin will need to be logged in and navigate to the admin home screen. In the admin home screen, there will be numerous tabs containing links to the various subsystems. The admin will have to click on the sales button to gain access to the sales management screen. On the Sales Management Screen the Admin will click on the View Details button to gain access the View Details of Sale Screen.

Once the Admin has clicked on the Open Invoice button a Sale Invoice will be generated according to the details of the client.
